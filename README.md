Сахалыы раскладка
------------------------------------------

### Описание
Якутская раскладка для пользователей Mac.
Совместим с Mac OS X 10.5 и выше.

### Установка
Для установки раскладки загрузите установщик.
После загрузки, откройте файл и следуйте по инструкции.

### Как пользоваться?
Для печати символа Һ нажмите одновременно клавишу Option и Ь, для других символов посмотрите ниже:
<ul>
<li>Һ — Option + Ь</li>
<li>Ө — Option + О</li>
<li>Ҕ — Option + Г</li>
<li>Ү — Option + У</li>
<li>Ҥ — Option + Н</li>
</ul>
